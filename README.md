# Calculadora de Interés Simple

## Descripción
Este proyecto es una calculadora de interés simple.

## Fórmula
Interés = Principal × Tasa × Tiempo

## Cómo usar
1. Ingresa el monto principal
2. Ingresa la tasa de interés anual
3. Ingresa el tiempo en años
4. Obtendrás el interés calculado
