# Guía de Contribución

¡Gracias por tu interés en contribuir a este proyecto!

## Tipos de contribuciones aceptadas

Se aceptan todas las contribuciones, incluyendo:

- **Informes de errores** - Reporta cualquier bug que encuentres
- **Correcciones de errores** - Soluciona problemas existentes
- **Mejoras en la documentación** - Ayuda a mejorar la claridad y completitud de la documentación
- **Mejoras** - Sugiere o implementa mejoras en el código
- **Ideas** - Comparte nuevas ideas para el proyecto

## Cómo contribuir

1. Haz un fork del repositorio
2. Crea una rama para tu contribución (`git checkout -b mi-contribucion`)
3. Realiza tus cambios
4. Haz commit de tus cambios (`git commit -m "Descripción de mi contribución"`)
5. Push a tu rama (`git push origin mi-contribucion`)
6. Abre un Pull Request

## Proceso de revisión

Todas las contribuciones serán revisadas por los mantenedores del proyecto. Se proporcionará retroalimentación y, si es necesario, se solicitarán cambios antes de la fusión.

## Código de Conducta

Al participar en este proyecto, aceptas seguir nuestro [Código de Conducta](CODE_OF_CONDUCT.md).
